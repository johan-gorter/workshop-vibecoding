# Source control

- Source control = een systeem dat **alle wijzigingen** in je code bijhoudt
- Vergelijk het met "Wijzigingen bijhouden" in Word, maar dan voor je hele project
- Je kunt altijd **terug naar een eerdere versie**
- Meerdere mensen kunnen **tegelijk** aan hetzelfde project werken
